# Zepto dashboard
Zepto Sales Ananlysis Dashboard.<br>
A Power BI dashboard that visualizes Zepto sales, customer trends, and inventory performance to support data-driven decisions.<br>

Description
Zepto Sales Analysis Dashboard is a Power BI project that analyzes sales performance, customer behavior, and inventory data. The dashboard provides clear insights using key metrics and visualizations to help understand trends, improve decision-making, and support business growth.<br>


The Dashboard was built using by using following tools and technologies:<br>
1.Power BI – Used to create interactive dashboards and visual reports<br>
2.Microsoft Excel / CSV – Used as the data source for sales and inventory data<br>
3.DAX (Data Analysis Expressions) – Used for calculations like total sales, profit, and KPIs<br>
4.Data Cleaning Techniques – Used to remove errors, handle missing values, and prepare data<br>
5.Data Visualization – Charts, graphs, and KPIs used to easily understand trends and patterns<br>

Business Impact<br>

Helped identify high-performing and low-performing products, enabling better stock planning.<br>
Improved understanding of sales trends across different time periods for smarter business decisions.<br>
Supported profit optimization by analyzing the impact of discounts on revenue.<br>
Enabled data-driven decisions through easy-to-understand visual dashboards.<br>
Reduced manual reporting effort by providing automated KPI tracking.<br>
