# Zepto dashboard
Zepto Sales Ananlysis Dashboard
A Power BI dashboard that visualizes Zepto sales, customer trends, and inventory performance to support data-driven decisions.

Description
Zepto Sales Analysis Dashboard is a Power BI project that analyzes sales performance, customer behavior, and inventory data. The dashboard provides clear insights using key metrics and visualizations to help understand trends, improve decision-making, and support business growth.


The Dashboard was built using by using following tools and technologies:
1.Power BI – Used to create interactive dashboards and visual reports
2.Microsoft Excel / CSV – Used as the data source for sales and inventory data
3.DAX (Data Analysis Expressions) – Used for calculations like total sales, profit, and KPIs
4.Data Cleaning Techniques – Used to remove errors, handle missing values, and prepare data
5.Data Visualization – Charts, graphs, and KPIs used to easily understand trends and patterns

Business Impact

Helped identify high-performing and low-performing products, enabling better stock planning.
Improved understanding of sales trends across different time periods for smarter business decisions.
Supported profit optimization by analyzing the impact of discounts on revenue.
Enabled data-driven decisions through easy-to-understand visual dashboards.
Reduced manual reporting effort by providing automated KPI tracking.
